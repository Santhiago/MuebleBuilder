# MuebleBuilder
Sitio Web para personalización de Muebles en línea

Integrantes: 
	*Erick Buendía
	*Edder Briones


Descripción:
Aplicación web que permita a los posibles compradores diseñar sus propios modelos de muebles a través de un prototipado 3D.
